Instructions On Installation

1.) Create a database in MySQL/phpmyadmin. Please make sure it is utf8_general_ci collation.

2.) Import the SQL file in sql/documentation.sql in the database you created in step 1.

3.) Edit the file in scripts/config-sample.php with your database credentials and URL. Please make sure there is a trailing / on your URL.

3.) Rename the file in step 3 to scripts/config.php.

You should be all set to hit your documentation page! Simple setup!


How To Use

When you go to your URL that you created for the application (wherever index.php sits). You will notice an "Admin Area" link at the top of the page. Click that and it will bring you to a simple form to fill out for endpoints or for your own company/personal information. It's extremely simple. Fill out the information for an endpoint and click "create". Once the endpoint is created you can open the endpoint within the main URL just by selecting it and click "edit" if you ever need to edit or delete it.
